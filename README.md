# Automated Gameplay Testing Tool
Automated Gameplay Testing Tool - это небольшой проект на Unity, созданный для демонстрации навыков автоматизации проверок и разработки вспомогательных инструментов.
Проект не является игрой - это тестируемая сцена с автотестовым ботом и системой проверки состояния.

## Цель проекта
Цель проекта - показать:
- умение работать с автоматизацией и системным подходом
- умение декомпозировать игровую механику
- понимание жизненного цикла тестирования
- навыки написания тестовой документации
- базовые навыки разработки на Unity (C#)
- умение воспроизводить и фиксировать дефекты

## Тестируемый сценарий
Автотестовый бот выполняет последовательность действий:
1. Появляется на стартовой позиции
2. Двигается по маршруту
3. Нажимает кнопку
4. Проверяет, что дверь открылась
5. Проходит через дверь
6. Заходит в зону захвата
7. Захватывает контрольную точку
8. Доходит до финиша
9. Система формирует отчёт

## Что проверяется автоматически
| Проверка | Описание |
|---|---|
| Перемещение | Бот проходит весь маршрут |
| Button Interaction | Кнопка нажимается |
| Door State | Дверь открывается |
| Capture Point | Точка захватывается |
| Finish | Бот достигает финиша |
| Stuck Detection | Бот не застревает |
| FPS | Средний FPS выше порога |
| Timeout | Тест завершается при зависании |
| Отчёт | Формируется JSON-отчёт |

## Пример отчёта
После выполнения автотеста создаётся JSON-файл:
```json
{
    "TestName": "Main gameplay route auto test",
    "IsPassed": false,
    "AverageFps": 183.8349151611328,
    "HasBotReachedFinish": true,
    "IsBotStuck": false,
    "IsButtonPressed": true,
    "IsDoorOpened": true,
    "IsCapturePointCompleted": false,
    "FailureReason": ""
}
```
