# BUG-002

## Summary
Бот не ориентируется на реальные данные об игре, а двигается по точкам, указанным отдельно от реального положения необходимых объектов

## Серьёзность
Minor

## Приоритет
Low

## Окружение
- Scene: `GameplayTestScene`
- Platform: Windows

## Предусловия
- GameplayTestScene включена
- Бот находится на стартовой позиции.
- Маршрут содержит все контрольные точки, при этом положение контрольных точек указано неверно.

## Шаги
1. Запустить сцену.
2. Пройти ботом по всем контрольным точкам
3. Дойти ботом до финиша
4. Обратить внимание на значения в отчёте

## Ожидаемый результат
Бот проходит по контрольным точкам, которые совпадают с реальными координатами тестируемых объектов

## Фактический результат
Бот не ориентируется на реальные данные об игре, а двигается по точкам, указанным отдельно от реального положения необходимых объектов
