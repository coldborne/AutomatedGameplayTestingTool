# BUG-003

## Summary
Состояние бота `IsBotStuck` не устанавливается в `true` при застревании

## Серьёзность
Major

## Приоритет
Medium

## Окружение
- Scene: `GameplayTestScene`
- Platform: Windows

## Предусловия
- GameplayTestScene включена
- Бот находится на стартовой позиции.
- Маршрут содержит все контрольные точки, при этом положение контрольных точек указано так, чтобы бот обязательно застрял.

## Шаги
1. Запустить сцену.
2. Пройти ботом до момента застревания
3. Тест проваливается досрочно и автоматически завершается
4. Обратить внимание на значение `IsBotStuck` в отчёте

## Ожидаемый результат
Состояние бота `IsBotStuck` устанавливается в `true` при застревании

## Фактический результат
Состояние бота `IsBotStuck` не устанавливается в `true` при застревании
