# BUG-001

## Summary
Бот не останавливается для захвата точки, а продолжает двигаться после достижения точки маршрута

## Серьёзность
Major

## Приоритет
Low

## Окружение
- Scene: `GameplayTestScene`
- Platform: Windows

## Предусловия
- GameplayTestScene включена
- Бот находится на стартовой позиции.
- Маршрут содержит точку захвата как контрольную точку.

## Шаги
1. Запустить сцену.
2. Переместить бота на зону захвата
3. Дождаться захвата зоны
4. Завершить движение бота, дойдя до финиша.

## Ожидаемый результат
Бот не выходит из зоны захвата, пока не захватит точку. Только после захвата бот продолжает движение по контрольным точкам

## Фактический результат
Бот не останавливается для захвата точки, а продолжает двигаться после достижения точки маршрута, не обращая внимание на захват точки.
