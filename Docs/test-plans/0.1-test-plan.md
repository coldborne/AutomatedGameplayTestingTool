# Тест-план v0.1

## 1. Цель
Реальный тест-план лучше оформлять в специализированной системе, например HP ALM, TestRail, Jira/Xray или аналогичном инструменте.  
В данном проекте используется Markdown-формат для удобства проверки прямо в GitHub.

Этот тест-план определяет область тестирования и набор тест-кейсов, которые необходимо выполнить для v0.1 проекта `Automated Gameplay Testing Tool`.
Цель - проверить, что автоматизированный бот корректно проходит игровой сценарий, взаимодействует с объектами сцены, 
обнаруживает некорректные состояния и формирует тестовый отчёт в формате json.

## 2. Объект тестирования
**Проект:** Automated Gameplay Testing Tool
**Версия:** v0.1
**Тип тестирования:** автоматизированное функциональное тестирование
**Платформа:** Windows

## 3. Область тестирования
В рамках тест-плана проверяются:
- запуск тестовой сцены
- движение бота по маршруту
- прохождение контрольных точек
- взаимодействие бота с кнопкой
- открытие двери после нажатия кнопки
- захват контрольной точки
- достижение финишной зоны
- обнаружение застревания бота
- автоматическое завершение теста при ошибке
- проверка среднего FPS
- формирование JSON-отчёта
- корректность итогового статуса теста

## 4. Критерии приёмки
Тестирование можно начинать, если:
- проект открывается в Unity без ошибок компиляции
- сцена `GameplayTestScene` запускается в Play Mode
- на сцене присутствует объект `GameplayBot`
- на сцене присутствует объект `GameState`
- на сцене присутствует объект `TestRunner`
- у бота настроены waypoint-точки маршрута
- объекты `PressureButton`, `Door`, `CapturePoint` находятся на сцене
- все ссылки в Inspector проставлены корректно
- папка `TestReports` доступна для записи.

## 5. Критерии завершенности
Тестирование считается завершённым, если:
- все тест-кейсы из текущего тест-плана выполнены
- для всех failed-сценариев оформлены баг-репорты или технические комментарии
- JSON-отчёты сформированы для выполненных автотестов
- отсутствуют blocker и critical дефекты
- результаты тестирования отражены в таблице выполнения
