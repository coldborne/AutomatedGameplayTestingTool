# Тест-кейс-006. Проверка застревания бота и корректное завершение автотеста

## Предусловия
- Сцена `GameplayTestScene` настроена.
- Компонент `StuckDetector` подключён.
- Установлены параметры:
  - `maximumStuckTime = 2 сек`
  - `minimumMovementDistance = 0.05`

## Шаги
1. Запустить сцену.
2. Пройти ботом до момента застревания
3. Тест проваливается досрочно и автоматически завершается
4. Обратить внимание на значение `IsBotStuck` в отчёте

## Ожидаемый результат
В отчёте указано:
```json
...
"IsPassed": false,
"IsBotStuck":true,
...
```
