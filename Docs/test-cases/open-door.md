## Тест-кейс-002. Открытие двери после нажатия кнопки

### Предусловие
- Сцена `GameplayTestScene` успешно открыта.
- На сцене есть кнопка.
- Кнопка связана с дверью.
- Бот находится на стартовой позиции.

### Шаги
1. Запустить сцену.
2. Переместить бота на кнопку
3. Дождаться открытия двери
4. Завершить движение бота, дойдя до финиша.

Ожидаемый результат
Кнопка нажата, дверь открыта. В отчёте указано:

```json
...
"IsButtonPressed": true,
"IsDoorOpened": true
...
```
