## Тес-кейс-001. Успешное прохождение основного маршрута ботом

### Предусловие
- Сцена `GameplayTestScene` успешно открыта.
- Бот находится на стартовой позиции.
- Маршрут содержит все контрольные точки.

### Шаги
1. Запустить сцену.
2. Дождаться завершения движения бота.
3. Проверить значение `HasBotReachedFinish` в отчёте.

### Ожидаемый результат
Бот достигает финиша. В отчёте указано:

```json
...
"HasBotReachedFinish": true
...
```
