## Тест-кейс-003. Захват контрольной точки

### Предусловие
- Сцена `GameplayTestScene` успешно открыта.
- Бот находится на стартовой позиции.
- На сцене есть зона захвата.

### Шаги
1. Запустить сцену.
2. Переместить бота на зону захвата
3. Дождаться захвата зоны
4. Завершить движение бота, дойдя до финиша.

### Ожидаемый результат
Точка захвачена. В отчёте указано:
```json
...
"IsCapturePointCompleted": true
...
```
