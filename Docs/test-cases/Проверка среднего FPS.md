## Тест-кейс-005. Проверка среднего FPS

### Предусловие
- Сцена `GameplayTestScene` успешно открыта.
- На сцене подключён FpsMonitor.
- Минимальный FPS установлен в 30.
- Бот находится на стартовой позиции.
- Маршрут содержит все контрольные точки.

### Шаги
1. Запустить сцену.
2. Пройти ботом по всем контрольным точкам
3. Дойти ботом до финиша
4. Проверить значение AverageFps в отчёте.

### Ожидаемый результат
Средний FPS не ниже 30. Тест не падает по FPS. В отчёте указано:
```json
...
"AverageFps": value
...
```
где value - это нецелое число не ниже 30.0
