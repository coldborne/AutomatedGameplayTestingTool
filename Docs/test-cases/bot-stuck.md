## Тес-кейс-004. Проверка отсутствия stuck-состояния

### Предусловие
- Сцена `GameplayTestScene` успешно открыта.
- На сцене подключён StuckDetector.
- Бот находится на стартовой позиции.
- Маршрут содержит все контрольные точки.

### Шаги
1. Запустить сцену.
2. Пройти ботом по всем контрольным точкам
3. Дойти ботом до финиша
4. Проверить значение IsBotStuck в отчёте.

### Ожидаемый результат
Бот не застрял. В отчёте указано:
```json
...
"IsBotStuck": false
...
```
